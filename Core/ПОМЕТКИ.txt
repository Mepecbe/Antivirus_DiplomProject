 Синтаксис команды 0*операнд1&операнд2
	Где 0 - идентификатор команды
		
	Идентификаторы команд
		0 - создать монитор для раздела <операнд1> с применением фильтра <операнд 2>
		1 - отключить(удалить) монитора для раздела <операнд1>

    Пример команды на включение мониторинга раздела D с отключенным фильтром 0*D:\&*.*
