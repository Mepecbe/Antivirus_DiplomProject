*RESERVE NEW FILE DETECTOR
 Синтаксис команды 0*операнд1&операнд2
	Где 0 - идентификатор команды
		
	Идентификаторы команд
		0 - создать монитор для раздела <операнд1> с применением фильтра <операнд 2>
		1 - отключить(удалить) монитора для раздела <операнд1>

    Пример команды на включение мониторинга раздела D с отключенным фильтром 0*D:\&*.*


сообщение во входящую трубу сервису сканирования
	id,file
	id - int, идентификатор задачи
	file - string, путь к файлу

ответ от сервиса скана с результатами сканирования
	id,result,virus
	id - int, идентификатор задачи 
	result - byte, результат (0 - нет, 1 - вирус)
	virus - int, идентификатор вируса

Труба сигнатур,  сервис скана <--> база сигнатур
	id,len,signature
	id - int16
	len - int16
	signature - byte[len]
